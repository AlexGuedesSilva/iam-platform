# IAM Platform

Initial Maven multi-module structure for a Java Spring Boot IAM Platform.

## Modules

- `iam-application`: main IAM application module.
- `api-gateway`: gateway module for routing and edge concerns.
- `shared-kernel`: shared domain-neutral contracts and utilities.

## Project Layout

```text
iam-platform/
├── iam-application/
├── api-gateway/
├── shared-kernel/
├── docs/
│   ├── architecture/
│   └── diagrams/
├── docker/
├── pom.xml
└── README.md
```

Business logic has not been implemented yet.
